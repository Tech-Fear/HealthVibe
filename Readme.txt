Running the HealthVibe Project Locally
	1. Navigate to the server directory: `cd server`
	2. Install required Node.js packages: `npm install`
	3. Install Python 3.10
	4. Install Python dependencies: `pip install -r requirements.txt`
	5. Install PyTorch, follow instructions at [PyTorch Official Website](https://pytorch.org/get-started/locally/)
	6. Start the server: `npm start`

Frontend:
	1. Navigate to the client directory: `cd client`
	2. Install required Node.js packages: `npm install`
	3. Start the frontend server: `npm start`
	4. Run the proxy server: `node proxy.js`
	5. The frontend will now be accessible and ready for use.
